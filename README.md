# Human Resources Analytics Dashboard

## Objective
Developed an interactive HR Analytics Dashboard using Power BI to analyze workforce trends, employee attrition, satisfaction, performance, and compensation.

## Dataset
HR Employee Attrition Dataset

## Tools Used
- Power BI Desktop
- Power Query
- DAX Measures

## Key KPIs
- Total Employees
- Attrition Rate
- Average Age
- Average Monthly Income
- Job Satisfaction
- Environment Satisfaction
- Performance Rating

## Visualizations
- Employees by Department
- Employee Attrition Analysis
- Gender Distribution
- Employees by Job Role
- Income by Department
- Overtime Analysis
- Work-Life Balance

## Key Insights
- Workforce trends analyzed
- Employee satisfaction evaluated
- Attrition patterns identified
- HR decision support dashboard

- HRAnalyticsDashboard.pbix
